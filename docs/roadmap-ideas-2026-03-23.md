## 3. 답변 품질 향상 (중기)

### 3-1. Chunk 품질 개선
- 현재 512자 고정 슬라이딩 윈도우 → **마크다운 헤딩 기반 시맨틱 청킹** 검토
- `## 제목` 단위로 자르면 맥락 유지 + 검색 정확도 ↑
- 트레이드오프: 청크 크기 불균일, 인덱스 사이즈 증가

### 3-2. Re-ranker 모델 도입
- 현재: 코사인 유사도 70% + BM25 30% (경량)
- 향후: Cross-encoder re-ranker (예: `ms-marco-MiniLM`) 추가 검토
- 효과: 검색 정확도 10-15% 향상 가능
- 트레이드오프: 추론 시간 +200ms, GPU 메모리 추가

### 3-3. 답변에 출처 인라인 표기
- 현재: 답변 하단에 출처 목록
- 개선: 답변 본문 중 `[1]`, `[2]` 같은 인라인 각주 → 클릭 시 원문 하이라이트
- Perplexity / ChatGPT Search 스타일

### 3-4. LLM 업그레이드
- 현재: Qwen3.5-9B-4bit (vLLM)
- 72B 모델 사용 시 답변 품질 대폭 향상 예상
- 추천 질문 생성 안정성도 개선 (현재 9B가 가끔 포맷을 안 지킴)
- GPU 메모리 요구: 40GB+ (A100/H100 필요)
