# 8단계 - 심사용 핵심 질문 세트 고정

## 목표

심사 직전에 반드시 돌려볼 핵심 질문 세트를 고정하고, 시연 가능한 수준인지 점검한다.

## 왜 이 단계가 마지막인가

앞 단계를 통해 기본 기능과 품질을 올린 뒤, 심사에서 직접 맞닥뜨릴 질문만 집중적으로 관리해야 한다.

## 담당 역할

- Ramanujan: 예상 질문군 최종 선정
- Arendt: 답변의 전달력과 안정감 점검
- Peirce: 질문별 기대 출처 검토
- Gibbs: 마지막 runtime 경로 점검
- Feynman: 실제 시연 순서대로 테스트
- Russell: README / 실행 경로 / 발표 동선 정리

## 작업 내용

심사용 핵심 질문 세트를 고정한다.

1. OCP 정의
2. 업그레이드 전 준비사항
3. disconnected 관련
4. operator 관련
5. 네트워크/스토리지/보안 질문
6. 5턴 이상 후속 질문

## 테스트

- 실제 시연 순서로 연속 실행
- citation 클릭 확인
- 멀티턴 유지 확인

## 완료 기준

- 시연용 질문 세트가 정리됨
- 각 질문에 pass/fail 과 리스크 메모가 남음
- 오늘 작업 마감 시점에 무엇이 준비됐는지 한눈에 설명 가능

## 산출물

- 최종 체크리스트
