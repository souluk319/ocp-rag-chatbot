# 1단계 - 기본 질문 구조 복구

## 목표

`오픈시프트가 뭐야`, `OCP가 뭐야`, `OpenShift가 뭐야` 같은 기본 질문에 한국어로 답하게 만든다.

## 왜 이 단계가 먼저인가

심사에서 가장 먼저 나올 질문일 가능성이 높고, 이 질문이 실패하면 프로젝트 전체 신뢰가 바로 무너진다.

## 담당 역할

- Ramanujan: 정의 질문 패턴 규칙 정리
- Arendt: 기본 개념 답변 문장 품질 정리
- Peirce: 공식 소개 문서 후보 확인
- Gibbs: gateway / policy / fallback 경로 구현
- Feynman: 실제 질문 재현 테스트
- Russell: 과한 우회 로직이 없는지 검토

## 작업 내용

1. 정의 질문 판별 규칙을 확정한다.
2. 공식 소개 문서로 연결되는 안전한 fallback 경로를 만든다.
3. 답변이 “자료를 못 찾음”으로만 끝나지 않게 한다.
4. 출처는 반드시 공식 문서 viewer 로 연결한다.

## 테스트

아래 세 질문으로 확인한다.

- `오픈시프트가 뭐야`
- `OCP가 뭐야`
- `OpenShift가 뭐야`

## 완료 기준

- 세 질문 모두 한국어 답변이 나온다.
- 최소 1개 이상의 공식 citation 이 붙는다.
- citation 을 누르면 viewer 경로가 열린다.
