# Product Brief

## 제품

`Play Book Studio`는 OpenShift 4.20 운영/학습 문서를 검색하고, 근거 기반 답변과 읽기 좋은 매뉴얼북을 함께 제공하는 제품이다.

핵심 산출물은 두 개다.

1. `RAG용 고품질 코퍼스`
2. `유저용 고품질 매뉴얼북`

## 현재 데모 범위

현재 시연 범위는 `OpenShift 4.20` 한정이다.

가능:
- 질문/답변
- citation 표시
- 오른쪽 참조 패널
- 문서 열기
- 일부 follow-up

불안정:
- 영어 fallback 고가치 문서의 완전한 한국어 편입
- 전 범위 개념형 질문 품질
- 공식 문서급 매뉴얼북 퍼블리싱 품질

판정:
- 현재는 `좁은 범위의 시연 제품`
- 아직 `전체 완성품`은 아님

## 앞으로 만들 앱

1. `Phase 1`
OpenShift 4.20 전용 플레이북 제품

2. `Phase 2`
고가치 영어 fallback 문서까지 포함한 4.20 코퍼스/매뉴얼북 완성

3. `Phase 3`
4.16~4.21 멀티버전 확장

4. `Phase 4`
다른 도메인 pack으로 확장 가능한 지식 제품

## 계획과 일정

### 단계 1. 시연 안정화
- 시연 질문 8~10개 고정
- 질문/답변/참조/문서열기 통과
- 불안정 기능 제외

목표:
- `좁지만 실제로 되는 시연 제품`

### 단계 2. 4.20 코퍼스 완성
- `machine_configuration`
- `operators`
- `monitoring`
- `backup_and_restore`
- `installing_on_any_platform`

목표:
- 위 문서들이 실제 한국어 코퍼스와 매뉴얼북으로 편입되는 것

### 단계 3. 매뉴얼북 품질 상승
- 코드블록
- 절차 카드
- note/warning
- 표
- 문서 위계/여백

목표:
- `공식 문서와 비교 가능한 수준의 읽기 경험`

### 단계 4. 멀티버전 확장
- version-aware crawler
- canonical 유지
- 코퍼스/매뉴얼북 동시 생성

목표:
- `4.16~4.21 대응`

## 시연에서 말할 문장

이 제품은 `OpenShift 4.20 운영/학습 플레이북`을 먼저 완성하는 단계다.  
현재는 질문-답변-근거-문서열기 흐름을 시연할 수 있고, 다음 단계는 고가치 영어 fallback 문서까지 한국어 코퍼스와 매뉴얼북에 편입하는 것이다.
