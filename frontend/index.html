각 단계별 용어 가이드

S
Session (세션)
대화 세션을 식별하고 이전 대화 이력을 불러오는 단계
session_id 세션 고유ID · history_turns 이전 대화 턴 수
R
Query Rewrite (질문 재작성)
"그거 삭제하려면?" 같은 맥락 의존 질문을 "OCP Pod를 삭제하려면?" 처럼 독립 질문으로 변환
original 원본 · rewritten 재작성 · query_type 유형 (troubleshooting/concept/procedure 등)
E
Embedding (임베딩)
질문 텍스트를 384차원 벡터로 변환하여 문서 검색에 사용
dim 벡터 차원 · cache_hit true면 캐시 사용으로 즉시 반환
C
Semantic Cache (시맨틱 캐시)
유사도 0.95 이상의 이전 질문이 있으면 LLM 호출 없이 캐시 답변 즉시 반환
semantic_cache_size 캐시 항목 수 · hit 시 이후 단계 전부 스킵
V
Vector Search (IVF 벡터 검색)
K-Means 클러스터 기반 IVF 인덱스에서 유사 문서 청크 검색
candidates 후보 수 · top_score 최고 유사도 · n_probe 탐색 클러스터 수
K
Reranking (하이브리드 재정렬)
벡터 점수 + BM25 키워드 점수를 질문 유형별 가중치로 합산하여 최종 순위 결정
semantic 벡터 유사도 · keyword BM25 점수 · score 최종 점수 (한국어 +0.08 부스트 포함)
X
Context Build (컨텍스트 구성)
검색 문서 + 인접 청크(±2) 확장 → LLM에 전달할 참고 문맥 구성 (최대 6000자)
context_length 문맥 길이 · num_documents 참고 문서 수
L
LLM Generate (LLM 응답 생성)
컨텍스트 + 질문을 LLM에 전달, 토큰 단위 스트리밍으로 답변 생성
tokens 생성 토큰 수 · tokens_per_sec 초당 속도 · answer_length 답변 길이
!
Pipeline Complete (완료)
전체 파이프라인 소요 시간. 응답 캐시 저장 여부 표시
cached_after 이 응답의 캐시 저장 여부
